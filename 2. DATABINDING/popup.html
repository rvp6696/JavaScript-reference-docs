Type of popup in Javascript

Alert(): It is pre-defined function used to give confirmation message to the user, with 1 ok button.

Confirm(): It is pre-defined function used to ask confirmation from the user to proceed further.

Inside confirm function we have two buttons mentioned below

"Ok" : It returns > true

"Cancel" : It returns > false

Prompt(): It is pre-defined function used to ask non boolean input from user

with Ok and Cancel button and the Cancel button will return a value null
