Class : The class in JS is a collection of member variables and member functions

Member Variable: The variables defined under constructor using this operator. A member variable is able to access by all methods of class. Ex: this.city

Member method/ function : All the user defined function of a class is called member function.

Constructor : It is a pre-defined function of a javascript class. Constructor is also called "lifecycle function" of a class Constructor get called automatically while object creation of a class.

Object of class : The Object is instance of a class.

extends: The extends keyword used to inherit feature from one class to another
